# Inventario_tkinter

En este proyecto libre desarrollé un sistema básico de "inventario".

Este proyecto permite usar Google Sheets como "base de datos" por lo que desde la interfaz Tkinter el usuario es capaz de:
<div>
    
    Añadir el nombre de un producto

    Añadir el precio de compra del producto

    Añadir el precio de venta final

    Mostrar los productos en el inventario
</div>

<div>
<p>Mediante la aplicación podras consultar o añadir productos, actualizar los datos y salir de forma "rapida" y "facil"<p> 
<p>Al añadir Google Sheets al proyecto y no una base de datos convencional nos aseguramos que la informacion este siepre en linea y acceder a las funcionalidades
    que una hoja de datos nos permite.</p>
</div>
Desafios del proyecto:

<div>

    Uso de console.cloud.google.com para el manejo de las Google APIs

    Uso de la librería Tkinter

    Uso de la librería gspread, una API para  Google Sheets.
</div>
